Text Formatting Tags in HTML

This is a paragraph of text. You can use various tags to format the text . Here are some examples:

Bold Text

You can make text bold using the strong tag or the b tag.

Italic Text

You can make text italic using the em tag or the i tag.

Underlined Text

You can underline text using the u tag.

Strikethrough Text

You can create strikethrough text using the s tag or the del tag.

Superscript and Subscript

You can create superscript text using the sup tag and subscript text using the sub tag.
